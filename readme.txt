AI-powered product description generator for WooCommerce. Create SEO-friendly descriptions with one click.

== Description ==

AI Product Description Generator helps WooCommerce store owners create compelling, SEO-optimized product descriptions in seconds using artificial intelligence.

= Key Features =

* One-Click Generation - Generate descriptions directly from product edit page
* Multiple AI Providers - Support for OpenAI (GPT-4o) and DeepSeek
* SEO Optimization - Descriptions include relevant keywords naturally
* Multi-Language - Generate in Chinese (Simplified) and English
* Multiple Styles - Professional and casual tones
* WooCommerce Integration - Works seamlessly with product attributes

= How It Works =

1. Install and activate the plugin
2. Go to WooCommerce > AI Descriptions and enter your API key
3. Edit any product and click "Generate Description"
4. Review and save the AI-generated description

= Requirements =

* WooCommerce 5.0 or higher
* PHP 7.4 or higher
* Valid API key from OpenAI or DeepSeek

== Installation ==

1. Upload the plugin files to /wp-content/plugins/ai-product-desc-generator/
2. Activate the plugin through the Plugins screen in WordPress
3. Go to WooCommerce > AI Descriptions to configure your API key
4. Start generating descriptions on any WooCommerce product page

== Frequently Asked Questions ==

= Which AI provider should I use? =

For Chinese content, we recommend DeepSeek for cost-effectiveness. For English, OpenAI GPT-4o-mini offers great quality at low cost.

= How much does it cost per generation? =

You pay the AI provider directly. Typical costs are $0.001-0.01 per product description depending on the model used.

= Is my API key secure? =

Yes, your API key is stored securely in the WordPress database and is only used to communicate with your chosen AI provider.

= Does this work without WooCommerce? =

The plugin is designed specifically for WooCommerce products. WooCommerce must be installed for full functionality.
